# Lost & Found Management System (C Language)

This is a simple console-based Lost & Found Management System developed using C programming language.

## Features
- Add lost items
- Add found items
- Search items
- Menu-driven program
- Uses file handling

## Concepts Used
- C Programming
- File Handling (fopen, fprintf, fscanf)
- Strings and strcmp()
- Loops and conditions

## How to Run
1. Compile the program:
   gcc lost_found.c -o lostfound

2. Run the program:
   lostfound
